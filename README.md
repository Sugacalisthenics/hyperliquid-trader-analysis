# hyperliquid-trader-analysis
An end-to-end Data Science project analyzing the impact of Crypto Fear &amp; Greed sentiment on Hyperliquid trader performance, featuring an interactive Streamlit dashboard and a Random Forest profitability predictor.
# 📈 Primetrade: Trader Performance vs Market Sentiment

## 🚀 Project Overview
This project explores the behavioral patterns of crypto traders on the Hyperliquid platform in relation to market sentiment (Fear & Greed Index). The goal was to identify how market emotions affect daily PnL, trade frequency, and risk appetite.

## 🛠️ Tech Stack
- **Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Visualizations:** Matplotlib, Seaborn
- **Web Framework:** Streamlit (for Interactive Dashboard)
- **Machine Learning:** Scikit-learn (Random Forest Classifier)

## 📊 Key Features
- **Data Pipeline:** Automated merging of historical trade data with daily sentiment scores.
- **Behavioral Analysis:** Comparison of 'Active' vs 'Casual' trader segments during Fear and Greed cycles.
- **Interactive Dashboard:** Real-time visualization of PnL metrics and trade sizes.
- **Predictive Modeling:** A machine learning model that predicts trader profitability based on daily market metrics.

## 💡 Strategic Insights
- Implemented a dynamic position sizing rule for risk mitigation during 'Extreme Fear' periods.
- Identified volatility thresholds for casual traders to improve platform retention.
